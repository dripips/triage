Why another helpdesk?

Most OSS helpdesks were built before the LLM era and hardcode "customer support" vocabulary. They don't bend to an IT helpdesk for a school, a bug tracker for 3 devs, or a complaint system for a vet clinic.

Triage bends to your team. Every ticket type gets its own custom fields, workflow states, and AI behaviour. Configure once — the system adapts.

What's inside

🤖

AI agents

OpenAI / Anthropic / YandexGPT / Ollama. Auto-categorize, suggest replies, summarize threads, sentiment analysis. Autonomous mode monitors chats in real time.

💬

Telegram-style chat

Real-time via Turbo Streams + Action Cable. Read receipts ✓✓, date separators, file attachments, clipboard paste. System events in chat feed.

🧾

Invoicing

Per-item discount + hidden surcharge. Price list integration. PDF print. Payment config: Stripe, YooKassa, Tinkoff.

🔔

Multi-channel notifications

In-app bell with badge, email (SMTP), Telegram bot, Slack webhook. Turbo Stream broadcast for real-time updates.

🔁

Custom workflows

AASM-like states defined per ticket type via JSON. Transitions, auto-assign, human-readable system comments.

🌐

Three locales

RU / EN / DE with full coverage. DB-backed per-key translation editor. Language management in settings.

👥

Staff / Customer

Role-based access. Customers see only their tickets and public messages. Internal notes hidden from customers.

🔐

External SSO

JWT-based login from any customer auth system. Per-company secret, claim mapping, auto-provision.

⚙️

11 settings sections

General, AI, Notifications, SSO, API, Knowledge Base, Price Lists, Payments, Chat, Languages, Translations.

AI that actually helps

Standard mode: pick a provider, enter API key, done. Advanced mode: custom endpoint, temperature, per-task prompts, budget cap. AI monitors every chat and suggests corrections when agents go off-track.

Auto-categorize

New ticket → AI assigns category + priority + confidence.

Suggest reply

AI drafts a reply based on ticket + KB + price list context.

Chat monitoring

Autonomous AI watches conversations, posts internal suggestions when needed.
