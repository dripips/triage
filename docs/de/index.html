Was steckt drin

🤖

KI-Agenten

OpenAI / Anthropic / YandexGPT / Ollama. Auto-Kategorisierung, Antwortvorschläge, Zusammenfassungen, Stimmungsanalyse.

💬

Telegram-Chat

Echtzeit über Turbo Streams + Action Cable. Lesebestätigungen ✓✓, Datums-Trenner, Dateianhänge.

🧾

Rechnungen

Rabatte + versteckter Aufschlag. Preislisten. PDF-Druck. Stripe / YooKassa / Tinkoff.

🔔

Benachrichtigungen

In-App-Glocke, E-Mail, Telegram, Slack. Turbo Stream Broadcast.

🔁

Workflows

AASM-Zustände pro Ticket-Typ via JSON. Übergänge, Auto-Zuweisung.

🌐

3 Sprachen

RU / EN / DE. Per-Key Übersetzungseditor. Sprachverwaltung in den Einstellungen.

Schnellinstallation

git clone https://github.com/dripips/triage.git
cd triage
bin/setup
bin/dev

Öffnen Sie http://localhost:3000. Login: admin@triage.local / password123.
