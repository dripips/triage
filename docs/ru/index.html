Универсальная система поддержки для любой индустрии — от IT до ветклиники. Чат, счета, AI-агенты, кастомные workflow. Rails 8 · Apple HIG · 3 языка.

Установить за 2 минуты ⭐ GitHub

Что внутри

🤖

AI-агенты

OpenAI / Anthropic / YandexGPT / Ollama. Авто-категоризация, подсказки ответов, резюме, анализ настроения. Автономный режим.

💬

Чат как в Telegram

Real-time через Turbo Streams + Action Cable. Прочитано ✓✓, разделители дат, вложения, вставка из буфера.

🧾

Счета

Скидки + скрытая наценка per-позиция. Прайс-листы. PDF. Stripe / ЮKassa / Тинькофф.

🔔

Уведомления

In-app колокольчик, Email, Telegram, Slack. Turbo Stream broadcast.

🔁

Кастомные workflow

Состояния per-типу тикета через JSON. Транзишены, авто-назначение.

🌐

3 языка

RU / EN / DE. Редактор переводов per-ключ. Управление языками в настройках.

Быстрая установка

git clone https://github.com/dripips/triage.git
cd triage
bin/setup
bin/dev

Откройте http://localhost:3000. Логин: admin@triage.local / password123.
